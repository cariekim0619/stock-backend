1. 자동연결

주토피아 주식 계좌 자동 연동 마법사가 서버에서 한국투자증권 KIS Open API 연동을 시작하고, 사용자가 직접 KIS 앱 인증 및 휴대폰 본인확인을 완료한 뒤, 발급될 App Key와 App Secret을 읽어 서버로 전송하여 계좌연동을 진행합니다.

2. 수동 입력

사용자가 한국투자증권 KIS Open API 페이지에서 직접 App Key와 App Secret을 확인한 뒤, 주토피아 챗봇에 직접 입력하여 계좌연동을 진행합니다.

3. 공통 고지

두 방식 모두 KIS App Key, App Secret, Access Token, 계좌 식별정보가 처리될 수 있습니다.
App Key와 App Secret은 계좌 조회 API 접근에 사용될 수 있는 중요한 인증정보이므로 타인에게 공유하지 마세요.자동연결에 동의하지 않아도 수동 입력 방식으로 계좌연동을 진행할 수 있습니다.
수동 입력도 원하지 않는 경우 계좌연동 없이 기본 서비스를 이용할 수 있습니다.
