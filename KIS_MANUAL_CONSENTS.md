1. 수집·이용 목적

KIS Access Token 발급
계좌연동 상태 확인
보유종목 조회
거래내역 조회
계좌 기반 리포트 제공
계좌연동 오류 대응 및 보안 관리

2. 수집·이용 항목

KIS 연동 환경
KIS App Key
KIS App Secret
KIS Access Token 및 만료시각
계좌번호 또는 계좌 식별정보
보유종목 정보
거래내역 정보
수동 입력 처리 로그

3. 보유·이용 기간

App Key와 App Secret은 계좌연동 유지 기간 동안 보관합니다.
계좌연동 해제 또는 서비스 탈퇴 시 App Key, App Secret, Access Token, 계좌 식별정보를 삭제합니다.
보유종목 및 거래내역 조회 결과는 리포트 제공 목적 달성 후 필요한 최소 기간만 보관합니다.

4. 동의 거부권 및 불이익

사용자는 수동 입력 방식의 계좌연동에 동의하지 않을 수 있습니다.
동의하지 않으면 보유종목 확인, 거래내역 조회, 계좌 기반 리포트 기능은 이용할 수 없습니다.
다만 주식 용어 사전, 주식 종목 리포트, 뉴스 커뮤니티 요약 등 기본 기능은 계속 이용할 수 있습니다.
