고객 이용약관

오픈 API 서비스 이용 약관(고객)

제정 2022. 8. 8.

제 1 조 (목적)

한국투자증권 주식회사(이하 “회사”라 한다)가 운영하는 ”KIS Developers 홈페이지(apiportal.koreainvestment.com, 이하 “포털사이트”라 한다)”에서 제공하는 오픈 API 서비스를 이용함에 있어 “회사”와 오픈 API 서비스를 이용하는 회사의 고객(이하 “고객” 이라 한다) 간의 권리, 의무 및 책임사항을 명확히 규정함을 목적으로 한다.

제 2 조 (용어의 정의)

① 본 약관에서 사용하는 용어의 정의는 다음 각 호와 같다.

1. “오픈 API 서비스”란 회사가 제공하는 내부의 서비스 및 데이터를 표준화된 API(Application Programming Interface) 형태로 이용 신청한 고객이라면 누구나 사용할 수 있도록 제공하는 서비스를 의미한다.

2. "고객"이란 회사 포털사이트 등에 접속하여 본 약관 등에 따라 회사와 이용계약을 체결하고 회사가 제공하는 오픈 API 서비스를 이용하는 회사의 계좌를 보유한 자를 의미한다.

3. "가입"이란 회사 홈페이지(securities.koreainvestment.com) 에서 제공하는 오픈 API 서비스 신청 양식에 고객의 정보를 기입하고, 본 약관에 동의하여 오픈 API 서비스 이용 계약을 완료시키는 행위를 의미한다.

4. "포털 아이디(ID)”란 포털사이트를 이용하는 고객의 식별과 오픈 API 서비스 이용을 위해 고객이 설정하고 회사가 승인한 문자와 숫자의 조합을 의미한다. (단, 포탈사이트에 한하여 사용되는 ID로 회사 홈페이지, HTS, MTS에서 사용하는 고객ID와는 별개이다.)

5. "포털 ID 비밀번호(PW)"란 포털사이트를 이용하는 고객과 포털 ID의 동일성을 식별하고 고객의 정보 등 비밀 보호를 위하여 고객이 설정하고 회사가 승인한 문자와 숫자, 특수문자의 조합을 의미한다. (단, 포탈사이트에 한하여 사용되는 비밀번호로 회사 홈페이지, HTS, MTS에서 사용하는 고객ID 비밀번호와는 별개이다.)

제 3 조 (적용범위)

본 약관은 고객이 회사의 오픈 API 서비스를 이용하여 처리하는 조회(시세정보, 계좌정보, 잔고, 거래내역 등), 매매주문업무 등에 대하여 적용된다.

제 4 조 (회사의 의무)

① 회사는 본 약관에서 특별히 규정한 사유 외에는 오픈 API 서비스를 지속적·안정적으로 제공하여야 한다. 

② 회사는 오픈 API 서비스 이용과 관련하여 알게 된 고객의 정보를 업무 이외의 목적으로 사용할 수 없다. 다만, 「금융실명거래 및 비밀보장에 관한 법률」 등 관계법령이 정하는 바에 따라 국가기관 등이 요청하는 경우에는 그렇지 아니하다.

③ 회사는 고객이 오픈 API 서비스에 대하여 이의를 제기하거나 확인을 요청하면 이를 신속히 처리하여야 한다.

제 5 조 (고객의 의무)

① 고객은 오픈 API 서비스 이용의 대가로 수수료를 납부하여야 한다. 수수료 납부에 대한 사항은 제11조 제2항에 따른다.

② 고객은 자신의 보안코드(앱키, 시크릿키)가 유출 및 도용되지 않도록 관리해야 하고, 제3자에게 대여, 위임, 양도 또는 본인 이외의 제3자에게 누설해서는 아니 된다.

③ 고객은 회사에서 제공하는 시세(국내주식, 해외주식, 국내선물/옵션 등)정보를 고객이 직접 개발한 프로그램 등 개인의 업무에 한하여 이용해야 하며, 제3자에게 제공해서는 아니 된다.

④ 고객은 본 약관에서 정한 사항을 준수하여야 하며, 회사가 약관 이행 여부를 점검하고자 할 경우 적극 협조하여야 한다.

제 6 조 (약관의 효력)

① 고객이 본 약관에 동의하고, 회사가 이를 승인함으로써 오픈 API 서비스 이용 계약은 성립하며, 이용 승인이 해지될 때 오픈 API 서비스 이용 계약은 해지된다.

② 제1항의 경우, 본 약관의 내용은 포털사이트에 게시함으로써 그 효력이 발생한다.

제 7 조 (이용 신청)

① 회사의 오픈 API 서비스를 이용하고자 하는 고객은 회사 홈페이지에서 약관 동의, 본인인증, 계좌인증 등을 통해 가입하여 오픈 API 서비스를 신청해야 한다.

② 고객이 오픈 API 서비스를 신청하면 회사는 보안코드(앱키, 시크릿키)를 발급하며 고객은 제5조 제2항에 따라 관리하여야 한다.

제 8 조 (이용 승인 불허)

① 회사는 고객이 다음 각 호의 어느 하나에 해당할 경우 이용신청에 대해 승인하지 아니한다.

1. 타인의 명의를 사용하여 신청한 경우

2. 오픈 API 서비스 이용 신청 내용을 허위로 기재하여 신청한 경우

3. 만19세 미만의 미성년자가 신청한 경우

② 제1항의 경우 회사는 고객에게 이용 승인 불가 사유를 통지한다. 

제 9 조 (이용 중지)

① 회사는 고객이 다음 각 호의 어느 하나에 해당할 경우 동 고객에 대한 오픈 API 서비스 이용을 중지할 수 있다. 

1. 수수료 납부계좌에서 부족분이 발생한 경우. 단, 수수료가 0원인 경우는 해당하지 않는다.

2. 금융감독당국으로부터 중지 권고가 있는 경우

3. 고객이 오픈 API 서비스 이용해 불법 대여 계좌를 운영하는 경우

4. 회사의 서버에 일정수준 이상의 부하를 유발하는 등 오픈API서비스의 안정적인 운영을 위하여 필요하다고 인정하는 경우

② 고객이 제1항에 해당할 경우, 회사는 해당 고객에게 소명 자료의 제출을 요구할 수 있으며, 고객은 통지를 받은 날로부터 10일 이내에 소명 자료를 제출하여야 한다. 다만, 회사는 고객의 소명자료가 정당하다고 판단되기 전까지는 오픈 API 서비스의 이용을 중지할 수 있다.

제 10 조 (이용 해지)

① 회사는 고객이 다음 각 호의 어느 하나에 해당할 경우 제2항 및 제3항의 절차에 의거 고객에게 최고하고 정당한 소명이 되지 않는 경우 이용 승인을 해지할 수 있다.

1. 타인의 명의를 사용하거나 허위 내용을 기재하여 신청한 경우

2. 고객 수수료 납부계좌에서 부족분이 3회 이상 발생한 경우.

3. 금융감독당국으로부터 해지 권고가 있는 경우

4. 최근 1년간 오픈 API 서비스 이용 실적이 없는 경우

5. 고객의 오픈 API 서비스 이용해 불법 대여 계좌를 운영하는 경우

6. 회사의 서버에 일정수준 이상의 부하를 유발하는 등 오픈API서비스의 안정적인 운영을 위하여 필요하다고 인정하는 경우

② 고객이 제1항에 해당할 경우, 회사는 해당 고객의 오픈 API 서비스 이용을 중지하고 소명 자료의 제출을 요구할 수 있으며, 고객은 통지를 받은 날로부터 10일 이내에 소명 자료를 제출하여야 한다. 다만, 회사는 고객의 소명자료가 정당하다고 판단되기 전까지는 본 오픈 API 서비스 이용을 중지할 수 있다.

③ 회사는 고객이 정당한 사유 없이 제2항의 소명에 응하지 아니하거나, 고객의 소재 불명으로 의견 진술의 기회를 줄 수 없는 등 해당 기간 내에 소명 내지 시정이 이루어지지 아니한 경우, 오픈 API 서비스의 이용을 해지할 수 있다.

제 11 조 (수수료)

① 회사는 오픈 API 서비스 이용 고객에게 오픈 API 서비스 제공 대가로 수수료를 부과한다. 단, 수수료가 0원인 경우는 해당하지 않는다.

② 수수료 납부 기준, 납부 절차 등 세부 사항은 회사가 정하여 회사 홈페이지 또는 포털사이트에 게시한다. 

제 12 조 (유량 제어)

① 회사는 원활한 오픈 API 서비스 제공을 위해 유량 제어 정책에 의거하여 고객의 API 요청을 제한한다.

② 유량 제어 정책은 회사가 정하여 회사 홈페이지 또는 포털사이트에 게시한다.

제 13 조 (운영시간)

오픈 API 서비스 운영시간은 연중무휴로 하며, 서비스 점검시간에는 거래내역이 시스템에 즉시 반영되지 않아 잔고 조회 등 일부 조회 사항이 실제와 다르게 조회될 수 있으며, 이 경우 회사는 이에 대해 고객에게 공지한다. 다만, 제 16조 오픈 API 서비스 변경 및 중지 사유인 경우는 운영을 중단할 수 있다

제 14 조 (거래기록의 보존기간, 방법 등)

① 회사는 오픈 API 서비스의 내용을 추적, 검사하거나 그 내용에 오류가 발생한 경우 이를 확인하거나 정정할 수 있는 정보를 10년간 유지, 보존하고 보존기간 경과 시 해당 거래기록을 파기한다.

② 회사는 제1항의 거래기록을 서면 또는 전산정보 저장매체를 이용하여 보존한다.

③ 회사는 고객의 요청이 있을 경우 「금융실명거래 및 비밀보장에 관한 법률」 등 다른 법률에 저촉되지 않는 범위 내에서 제1항의 거래기록을 제공할 수 있다.

제 15 조 (약관의 변경)

① 회사는 약관을 변경하고자 하는 경우 시행일 1개월 전에 변경되는 약관을 전자금융거래를 수행하는 전자적 장치(해당 전자적 장치에 게시하기 어려운 경우에는 고객이 접근하기 쉬운 전자적 장치로서 회사가 지정하는 대체장치를 포함한다. 이하 이조에서 같다.)에 게시하고 고객에게 전자문서 등 고객과 사전에 합의된 방법으로 통지하여야 한다. 다만, 고객이 이의를 제기할 경우 회사는 고객에게 적절한 방법으로 약관 변경내용을 통지하였음을 확인해 주어야 한다.

② 제1항에 불구하고 회사가 법령의 개정으로 인하여 긴급하게 약관을 변경하는 때에는 변경된 약관을 전자적 장치에 1개월 이상 게시하고 제1항에서 정한 방법으로 고객에게 통지하여야 한다.

③ 고객은 제1항 및 제2항에 따라 약관의 변경내용이 게시되거나 통지된 후부터 변경되는 약관의 시행일 전의 영업일까지 전자금융거래계약을 해지할 수 있다.

④ 회사는 제3항에 따른 기간동안 고객이 약관의 변경내용에 대하여 이의를 제기하지 아니하는 경우에는 약관의 변경을 승인한 것으로 본다.

⑤ 회사는 고객이 약관의 교부를 요청하는 경우 이에 응하여야 하며, 전자금융거래를 수행하는 전자적 장치에 게시하여 고객이 약관을 조회하고 다운로드(화면출력을 포함한다)받을 수 있도록 한다.

제 16 조 (서비스 변경 및 중지)

① 회사는 수시로 오픈 API 서비스의 향상을 위해 오픈 API 서비스의 전부 또는 일부 내용을 변경할 수 있다. 이 경우 회사는 변경될 오픈 API 서비스의 내용 및 제공 일자를 포털사이트에 게시하여 변경 15일 전에 고객에게 공지한다.

② 회사는 다음 각 호에 해당하는 경우 오픈 API 서비스의 전부 또는 일부를 제한하거나 중지할 수 있다.

1. 컴퓨터 등 정보통신설비의 보수점검, 교체 및 고장, 통신의 두절 등의 사유가 발생한 경우

2. 오픈 API 서비스를 위한 설비의 보수 등 공사로 인해 필요한 경우

3. 오픈 API 서비스 업그레이드 및 시스템 유지보수 등을 위해 필요한 경우

4. 정전, 설비의 장애 또는 이용량의 폭주 등으로 정상적인 오픈 API 서비스 이용에 지장이 있는 경우

5. 고객이 회사의 오픈 API 서비스 운영을 방해하는 경우

6. 회사의 경영상의 판단 등 회사의 사정으로 오픈 API 서비스를 유지할 수 없는 경우

7. 기타 천재지변, 국가비상사태 등 불가항력적 사유가 있는 경우

③ 제2항에 의하여 오픈 API 서비스가 제한 또는 중지될 경우, 회사는 이를 포털사이트 초기화면에 게시하는 방법으로 고객에게 공지한다. 다만, 회사가 통제할 수 없는 사유로 오픈 API 서비스가 중단(회사 또는 운영자의 고의 및 과실이 없는 디스크 장애, 시스템 다운 등)되어 사전 공지가 불가능한 경우에는 그렇지 아니하다.

④ 전산시스템 또는 네트워크, 전화회선 등에 장애가 발생하여 주문 입력 등 업무처리가 불가능한 경우에는 전자금융거래 담당부서에서 고객의 전화 신청에 의해 업무처리를 대행한다.

1. 회사의 고객센터로 전화주문 ( 전화 : 1588-0012, 1544-5000 )

2. 본사 영업부 및 영업점 방문 ( 문의전화 : 02-3276-5050 )

제 17 조 (손해배상 및 면책)

① 회사 또는 고객이 의무를 이행하지 아니하여 상대방에게 손해를 발생시켰을 때에는 이를 배상하여야 한다. 

② 회사는 오픈 API서비스 이용과 관련하여 회사의 책임 있는 사유가 없는 경우로서 다음 각 호의 어느 하나에 해당하는 사유로 인하여 고객에게 발생하는 손해에 대하여 책임을 지지 아니한다.

1. 고객의 고의 또는 과실에 의한 인증키(앱키, 앱시크리트, 접근토큰 등) 유출로 비롯한 고객 이외의 자에 의한 부정 사용의 경우

2. 부정 사용의 피해조사를 위하여 회사가 정한 조사에 협조하지 아니한 경우

3. 천재지변 또는 이에 준하는 불가항력으로 오픈 API 서비스를 제공할 수 없는 경우

③ 제3자의 부정사용 및 접근매체의 분실, 도난의 경우 고객이 자신에게 제2항 제2호의 귀책사유가 없음을 입증하지 않는 한 그로 인한 손해에 대하여 회사는 책임을 지지 않는다.

④ 회사는 오픈 API 서비스의 변경 또는 정지로 인해 발생되는 문제에 대해 고의나 중과실이 없는 이상 어떠한 책임도 지지 않는다.

제 18 조 (이의 제기)

① 고객은 오픈 API 서비스의 처리와 관련하여 이의가 있을 때에는 서면으로 회사에 해결을 요구하거나 분쟁처리를 신청할 수 있다.

② 고객이 회사에 이의를 제기한 경우 회사는 이에 대한 조사 또는 처리결과를 14일 이내에 고객에게 통지하여야 한다.

③ 회사는 분쟁 처리 책임자 및 담당자를 지정하고, 그 연락처를 관련 회사의 포털사이트에 게시하여야 한다.

④ 고객은 분쟁 처리와 관련한 회사의 사고조사 및 관계당국의 수사에 협조하여야 한다.

제 19 조 (분쟁 조정 및 관할 법원)

① 회사와 고객은 오픈 API 서비스와 관련하여 분쟁 발생시 원만하게 해결하기 위하여 상호 협조하여야 한다.

② 제1항의 규정에도 불구하고 본 오픈 API 서비스 이용과 관련한 분쟁으로 소송이 제기될 경우 관할법원은 민사소송법에서 정하는 바에 따른다.

제 20조 (다른 약관과의 관계)

① 본 약관 외에도 「전자금융거래이용에관한기본약관」이 적용되며, 규정된 내용이 서로 다를 경우 [전자금융거래이용에관한기본약관] 제8조를 제외하고 본 약관의 규정이 우선한다.

② 본 약관과 「전자금융거래 이용에 관한 기본 약관」에서 정하지 않은 사항에 대하여 다른 약정이 없으면 「전자금융거래법」, 「금융실명거래 및 비밀보장에 관한 법률」 등 관계법령을 적용한다.

이용약관 개정 알림

(필수) 이용약관 개정 시 메일/알림톡 받기
	
약관 개정 시 이메일(또는 카카오톡 알림톡)로 안내드리며 당사 홈페이지(오픈API포털 "KIS Developers")공지사항에 게시함으로 고객에게 통지를 완료한 것으로 합니다.

당사 제공 샘플코드 관련 유의사항

(필수) 당사에서 제공하는 샘플코드(KIS Developers 포털, 한국투자증권 Github)에 대한 유의사항 확인

샘플 코드는 한국투자증권의 오픈API(KIS Developers)를 연동하는 예시입니다. 고객님의 개발 부담을 줄이고자 참고용으로 제공되고 있습니다.

샘플 코드는 별도의 공지 없이 지속적으로 업데이트될 수 있습니다.

샘플 코드를 활용하여 제작한 고객님의 프로그램으로 인한 손해에 대해서는 당사에서 책임지지 않습니다.

Open API 시세 관련 유의사항

(필수) 당사 Open API 시세 이용 시 아래 유의사항을 모두 확인하였으며, 이에 동의합니다.

당사 Open API 시세정보는 한국투자증권 계좌를 보유한 개인 고객이 자기 자산의 투자 목적에 한해 이용할 수 있으며, 제3자 제공은 불가합니다.

법인(제휴사 포함)은 API를 통해 시세정보를 이용하려면 코스콤 등 거래소와 정보이용계약을 반드시 체결해야 합니다.
계약 없이 시세정보를 수신하거나 사용하는 경우, 발생하는 모든 법적 책임은 해당 법인이 부담하며, 거래소 정책 위반 시 API 서비스 이용이 제한될 수 있습니다.
